== Description ==

## Checkout Field Builder (Checkout Field Editor & Manager) for WooCommerce

**Checkout Field Builder (Checkout Manager) for WooCommerce** is the ultimate **checkout field editor** and **checkout field manager** plugin designed to enhance your WooCommerce checkout process. This powerful tool enables you to effortlessly **customize checkout fields** on your WooCommerce checkout page, offering complete control over **billing**, **shipping**, and **additional fields** sections. With support for over 20 different field types, you can easily **add, remove, reorder**, and **customize WooCommerce checkout fields** to meet your store's unique requirements.

### Key Features

- **Add, Edit, or Remove Custom Fields**: Modify the default WooCommerce fields or create entirely new custom fields using various field types, including **text, textarea, password, select, radio, checkbox, file upload, timepicker, datepicker,** and more.
- **20+ Custom Field Support**: Support for 20 field types, including text, dropdowns, checkboxes, and more.
- **Enable and Disable Fields Anytime**:Enable or disable fields..
- ** Ability to Displace Checkout field in Order Details**:Display custom checkout fields in emails and order details.

### Supported WooCommerce Checkout Field Types

Checkout Field Builder (Checkout Manager) for WooCommerce supports a wide range of field types to help you **customize WooCommerce checkout forms** according to your store's specific needs:

1. **Text**: Standard text input field for entering single-line text.
2. **Email**: Field for capturing email addresses.
3. **Number**: Input for numerical values, such as quantity or measurements.
4. **Country**: Dropdown for selecting a country.
5. **State**: Dropdown for selecting a state or region.
6. **Telephone**: Field for entering phone numbers.
7. **Hidden**: Invisible fields for storing data that doesn't need to be displayed to users.
8. **Password**: Input for entering passwords securely.
9. **Textarea**: Multi-line text input for longer messages or descriptions.
10. **Radio**: Radio buttons for selecting one option from a list.
11. **Checkbox**: A checkbox input for binary choices (yes/no, agree/disagree).
12. **Select**: Dropdown list for selecting one option.
13. **Multiselect**: Dropdown list allowing selection of multiple options.
14. **Checkbox Group**: Group of checkboxes for multiple selections.
15. **Heading**: Section heading to organize fields.
16. **Paragraph**: Informational text or message.
17. **URL**: Input for entering a web address.
18. **Datetime Local**: Field for selecting both date and time.
19. **Date**: Input for selecting a date.
20. **Time**: Input for selecting a time.
21. **Month**: Dropdown to select a month.
22. **Week**: Input for selecting a week within a year.

== Installation ==

### Automatic Installation

1. Log in to your WordPress dashboard.
2. Navigate to the "Plugins" menu and click "Add New".
3. Search for 'Checkout Field Builder' plugin.
4. Click 'Install Now', and WordPress will handle the rest.

### Manual Installation

Manual installation requires downloading the 'Checkout Field Builder' plugin and uploading it to your web server via FTP. Follow the [WordPress manual plugin installation guide](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation) for detailed instructions.
